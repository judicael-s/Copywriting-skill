# SEO growth copy integration

Use this to connect the copywriting suite with SEO growth workflows.

## Principle

SEO copy should not only rank. It should move demand toward an action.

Use analytics and search evidence before writing when available.

## Evidence stack

| Source | What it tells you |
|---|---|
| Google Search Console | queries, impressions, clicks, CTR, positions |
| GA4 | sessions, engagement, conversions, revenue |
| Google Trends | seasonality and relative interest |
| DataForSEO or Ads | volume, CPC, difficulty, SERP data |
| SERP review | intent, format, competitors, weak results |
| Public crawl | metadata, headings, internal links, indexability |

## Content brief additions

Add these to SEO copy briefs:

- target query and intent
- SERP format to match or beat
- GSC page/query evidence if available
- GA4 landing-page behavior if available
- Google Trends seasonality if relevant
- internal link targets and anchor text
- CTA/product mapping
- proof points with provenance
- post-publish measurement window

## CTA and product mapping

Every SEO page should know what it is meant to move the reader toward.

Examples:

- informational article -> related guide, diagnostic, newsletter, free tool
- commercial article -> comparison, product, service page, consultation
- product/category page -> purchase, quote, demo, bundle
- local page -> call, booking, directions, quote

## Post-publish loop

After publishing, check:

- 7 days: indexation and basic tracking
- 28 days: GSC clicks/impressions and GA4 engagement
- 60 days: ranking movement and conversion behavior
- 90 days: keep, refresh, merge, expand, or redirect

## Output rule

When drafting SEO copy, include:

- copy
- metadata
- internal-link plan
- CTA plan
- measurement plan
- refresh date
